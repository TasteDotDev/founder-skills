# Founder Skills

530 business frameworks as AI-powered skills. Works with Claude Code, OpenAI GPTs, Gemini Gems, and the [Founder CLI](https://github.com/TasteDotDev/founder-cli).

## Structure

```
skills/
  strategy/          # Porter, Blue Ocean, Wardley Maps, ...
  marketing/         # Growth loops, positioning, PMF, ...
  sales/             # MEDDIC, Challenger, SPIN, ...
  finance/           # Unit economics, fundraising, ...
  leadership/        # Management frameworks
  operations/        # OKRs, lean, six sigma, ...
  innovation/        # Jobs-to-be-done, design thinking, ...
  decision-making/   # Mental models, risk analysis, ...
  communication/     # Storytelling, presentations, ...
  negotiation/       # BATNA, anchoring, ...
  psychology/        # Behavioral economics, biases, ...
  mental-models/     # First principles, inversion, ...
  productivity/      # GTD, deep work, ...
  organization/      # Org design, culture, ...
  design/            # Design thinking, UX, ...
  founder/           # Meta-skill: routes to the right framework
  commands/          # Shortcut commands
```

## Usage

### Claude Code

```bash
claude install github:TasteDotDev/FounderSkills
```

Then use `/founder` in any conversation.

### CLI

```bash
npm i -g @taste.dev/founder
founder "how do I price my SaaS?"
```

See [founder-cli](https://github.com/TasteDotDev/founder-cli) for more.
